# Module 3: NVIDIA Isaac - Weekly Progression

## Module Overview
Module 3 introduces students to NVIDIA's Isaac ecosystem, including Isaac Sim for high-fidelity simulation, Isaac ROS for hardware-accelerated perception, and Nav2 for navigation. This module builds on ROS 2 and simulation fundamentals from previous modules and prepares students for VLA systems in Module 4.

## Week 1: Introduction to Isaac Sim

### Topics Covered
- Isaac Sim installation and Docker setup
- Omniverse connection and USD basics
- Creating basic simulation environments in Isaac Sim
- Importing robot models into Isaac Sim
- Basic physics and rendering concepts

### Learning Objectives
- Install and configure Isaac Sim using Docker
- Understand USD (Universal Scene Description) basics
- Create simple simulation scenes in Isaac Sim
- Import and configure robot models in Isaac Sim
- Understand the differences between Isaac Sim and Gazebo

### Prerequisites
- Completed Modules 1 and 2 (ROS 2 and simulation fundamentals)
- Understanding of URDF robot modeling
- Basic knowledge of GPU computing concepts
- Docker familiarity (helpful but not required)

### Activities
- Install Isaac Sim using Docker
- Connect to Omniverse and explore sample scenes
- Import a simple robot model into Isaac Sim
- Create a basic simulation environment

### Assessment
- Successfully run Isaac Sim with Docker
- Demonstrate robot model import and basic simulation
- Compare Isaac Sim with Gazebo simulation approaches
